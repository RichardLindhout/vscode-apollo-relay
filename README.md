Simple configuration of vscode-apollo for Relay projects.

[Changelog](https://github.com/alloy/vscode-apollo-relay/blob/master/CHANGELOG.md)

## Install

```sh
# using npm
npm install --save vscode-apollo-relay

# using yarn
yarn add vscode-apollo-relay
```
